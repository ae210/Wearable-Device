# Wearable-Device
# イチゴ収穫・選別支援ウェアラブルデバイス（プロトタイプ）

農業の人手不足という課題に対し、「直感的な動作のみで、収穫と同時に規格判定ができる仕組み」をゼロから企画・開発しました。

## 概要
親指に装着した磁石と、人差し指に装着したIMUセンサー（慣性計測装置）間の磁気変化を利用し、果物を掴んだ際の指の開き具合からサイズ（重量・等級）を非破壊でリアルタイムに推定するシステムです。

##  システム構成と使用技術
ハードウェアの設計から、Wi-Fiのない農場環境を想定したクラウド通信・データ可視化までを一貫して構築しました。

* **マイコン・センサー:** ESP32-WROOM-32E / XIAO nRF52840 / BNO055 / BNO085
* **通信モジュール:** SIM7080G
* **バックエンド・解析:** Python / Flask
* **フロントエンド (Webダッシュボード):** HTML / CSS
* **インフラ:** Render

##  現場での実証実験（実地検証）
机上の空論で終わらせず、実際の現場環境で動作させることにこだわりました。
青森県弘前市の農場まで直接足を運び、プロトタイプの実証実験を実施。農作業特有の動作ノイズや装着感の課題を洗い出し、センサーの無線化などアジャイルな改善を繰り返しています。
